1. Explainable AI (XAI) - Word Highlighting
Konsep: Tunjukkan kenapa model memilih genre tertentu. Saat hasil keluar, highlight kata-kata spesifik di dalam teks deskripsi (misal: "sword", "explore") yang memiliki bobot TF-IDF tertinggi untuk genre tersebut.

Nilai Jual: Menunjukkan pemahaman ML tingkat lanjut (interpretability). Anda tidak hanya membuat black-box AI, tapi sistem analitik berbasis teks yang transparan.

2. Radar Chart untuk Multi-Label (UI/UX)
Konsep: Ganti atau lengkapi progress bar linear dengan grafik jaring laba-laba (Radar/Spider Chart) menggunakan library ringan seperti Recharts atau Chart.js.

Nilai Jual: Sangat ideal untuk merepresentasikan data Multi-Label. Tampilannya jauh lebih futuristik, analitis, dan padat informasi tanpa memerlukan elemen visual gambar cover game sama sekali.

3. Batch Processing via CSV (Automation)
Konsep: Tambahkan tab atau tombol untuk mengunggah file .csv berisi puluhan atau ratusan deskripsi game sekaligus. Backend FastAPI akan memproses semuanya dan mengembalikan file hasilnya.

Nilai Jual: Mendemonstrasikan efisiensi algoritma dan skalabilitas backend Anda dalam menangani otomatisasi pemrosesan data secara masal, sebuah nilai plus yang besar di lingkungan enterprise.
